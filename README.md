# PC-pdf-
pc端pdf古籍阅读器

这个比较鸡肋，电脑阅读pdf的软件太多了，而且用电脑单纯阅读古籍的也比较少。软件直接打包所有库，以免不同的电脑用不起，软件比较大。

电脑端是宽屏，增加双页阅读功能
电脑可以使用键盘，增加键盘调用
电脑打开文件方式多样，增加适配

因为windows端使用python编写，android端使用java编写，代码不通用，版本号单独计算。软件为自愿使用，自行承担使用带来的后果，有源代码的可以编译自用，禁止发布、售卖以及修改发布、售卖本软件。
